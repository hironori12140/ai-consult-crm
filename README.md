# AIコンサル案件管理 - 音声で自動更新

表を見ながら**話すだけ**で、各社の状況（営業中/コンサル中/アフターフォロー中）と最終アクションが自動更新されるツールです。

## 特徴

- **音声入力**：マイクボタンを押して話すと、その内容が自動で文字起こし
- **自然言語で更新**：プルダウンや手入力不要。「〇〇会社はコンサル中で、先週提案書を送った」と話すだけでOK
- **AI解析（オプション）**：設定でOpenAI APIキーを登録すると、GPTで高精度に解析。未設定の場合はルールベースで解析
- **ローカル保存**：ブラウザのlocalStorageに保存。CSVのインポート/エクスポートも可能

## 使い方

1. `index.html` をブラウザで開く（Chrome推奨・音声認識対応）
2. （任意）**⚙️ 設定**からOpenAI APIキーを入力すると、AI解析が有効になります
3. 案件一覧に会社を追加（「+ 新規追加」またはCSV取り込み）
3. **「話して更新」**ボタンを押して、各社の状況を話す
   - 例：「株式会社サンプルAは今コンサル中で、先週提案書を送った」
   - 例：「サンプルBは営業中、サンプルCはアフターフォローで導入完了」
4. **「解析して更新」**をクリック → 自動でテーブルが更新される

## 話し方のコツ

- **会社名**をはっきり言う（一覧に載っている会社名を含めて話す。既存の案件のみ更新され、新規追加はされません）
- **ステータス**：「営業中」「コンサル中」「アフターフォロー中」を含めて話す
- **最終アクション**：「〇〇を送った」「〇〇を実施した」など具体的に
- **今後やるアクション**：「次は〇〇」「来週〇〇」「今後〇〇」と話すと反映されます

├── index.html   # メイン画面

## 注意事項

- 音声認識はChrome/Edgeなど対応ブラウザで利用可能（要HTTPSまたはlocalhost）
- OpenAI APIキーはブラウザのlocalStorageに保存されます（個人利用想定）
- APIキー未設定時はルールベースで解析。設定時はGPT（gpt-4o-mini）で解析
- データはブラウザに保存されるため、別PCではCSVエクスポート→インポートで移行してください
